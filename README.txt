# Healthcare SQL Analysis Project

## 📊 Overview
This project focuses on analyzing healthcare data using SQL with a star schema design.

## 🧱 Data Model
- Fact Tables:
  - fact_claims
  - fact_patient_encounters

- Dimension Tables:
  - dim_patient
  - dim_provider
  - dim_date
  - dim_facility
  - dim_diagnosis

## 🔍 Key Analysis
- Revenue analysis
- Top providers per year
- Patient encounters analysis
- Cost analysis per diagnosis
- Percentage contribution

## 🛠 Tools
- SQL Server
- Excel (for validation)
